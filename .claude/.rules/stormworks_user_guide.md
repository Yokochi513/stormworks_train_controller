# Stormworks Lua スクリプト ユーザーガイド作成規約

## 対象読者

このシステムを導入しようとするすべての Stormworks ユーザー。Lua の知識は前提としない。

---

## ファイル配置

- スクリプトファイルと**同階層**に `README.md` として配置する。
- 例: `controller/single_car.lua` → `controller/README.md`

---

## 必須セクションと順序

以下の順序で記載する。

1. **概要** — このスクリプトが何をするか（1〜2行）
2. **対象車種** — TA / TB / TAB のどれに使うか
3. **プロパティ設定** — マイコン内で設定が必要な `property` の一覧
4. **入力チャンネル一覧** — コンポジット入力の定義
5. **出力チャンネル一覧** — コンポジット出力の定義
6. **接続手順** — どのコネクタに何を繋ぐか（ASCII図付き）
7. **操作手順** — ゲーム内でどう使うか（ボタン操作ごとの箇条書き）
8. **注意事項** — よくある失敗・誤接続時の挙動など

---

## 各セクションの記述形式

### チャンネル一覧（入力・出力共通）

Markdown テーブル形式で記載する。`requirements.md` のチャンネル定義と列構成を統一する。

**Number チャンネル**

```markdown
| CH | 型     | 用途               | 値域      |
|----|--------|--------------------|-----------|
| 1  | Number | スロットルレバー入力 | 0.0〜1.0 |
```

**Bool チャンネル**

```markdown
| CH | 型   | 用途             |
|----|------|------------------|
| 1  | Bool | 右ドアボタン     |
```

### プロパティ設定

デフォルト値を含む Markdown テーブルで記載する。

```markdown
| ラベル  | 型   | 説明              | デフォルト |
|--------|------|-------------------|-----------|
| Is TA  | Bool | 先頭車なら true   | false     |
| Is TAB | Bool | 最後尾車なら true | false     |
```

### 接続手順

ASCII 図でコネクタの向きと接続先を示す。

```markdown
[前方コネクタ] ── [前隣車 / TB]
      |
  [マイコン]
      |
[後方コネクタ] ── [後隣車 / TB]
```

### 操作手順

ボタンごとに「チャンネル番号 + 動作結果」をセットで箇条書きする。

```markdown
- マスターリクエストボタン（Bool IN CH5）: 押すたびにマスター ON/OFF が切り替わる
- パーキングブレーキボタン（Bool IN CH6）: 押すたびに PB ON/OFF が切り替わり、ON 時はブレーキ 100% に固定される
```
