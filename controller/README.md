# 列車コントローラー ユーザーガイド

`controller/` ディレクトリのマイコン用スクリプトの導入・操作ガイドです。本ディレクトリには次の 2 つのスクリプトが含まれます。

| スクリプト | 用途 |
|-----------|------|
| `single_car.lua` | **一両編成専用**。TA + TAB を 1 マイコンで兼ね、車間通信を行わない。 |
| `car_logic.lua` | **汎用（ロジック）**。TA / TB / TAB / 一両車に対応し、車間通信でマスターと命令を編成全体へ伝播する。 |
| `car_display.lua` | **汎用（表示）**。`car_logic.lua` の出力を受け取り運転台モニタへ描画する専用チップ。 |

> `car_logic.lua` はマイコンの 8192 バイト上限を超えたため、ロジックと表示を **2 つのマイコンに分割**しています。汎用構成では `car_logic.lua` と `car_display.lua` を**必ずペアで**使用します（表示が不要なら `car_display.lua` は省略可）。

## スクリプトの選び方

- **単一の一両車だけを動かす** → `single_car.lua`（設定不要・最小構成・表示込み）。
- **複数両編成、または将来連結する可能性がある** → `car_logic.lua` ＋ `car_display.lua`（プロパティで車種を指定）。

> `car_logic.lua` は `Is TA` と `Is TAB` を両方 true にすると一両車として動作します。

# 1. single_car.lua — 一両車コントローラー

## 概要

TA（前部運転台）と TAB（後部運転台）の機能を 1 つのマイコンで兼ねる、**一両編成専用**の列車コントローラーです。
スロットル・ブレーキ・ドア・ライト・パーキングブレーキ・バック（逆転）を制御します。

## 対象車種

| 車種 | 説明 |
|------|------|
| 一両車 | 先頭・最後尾を兼ねる単体車両（TA + TAB） |

> 複数両編成（TA / TB / TAB を分ける構成）には使用できません。その場合は `car_logic.lua`（＋ `car_display.lua`）を使用してください。

## プロパティ設定

このスクリプトに設定が必要なプロパティはありません。

## 入力チャンネル一覧

**Number チャンネル**

| CH | 型     | 用途               | 値域      |
|----|--------|--------------------|-----------|
| 1  | Number | スロットルレバー入力 | 0.0〜1.0 |
| 2  | Number | ブレーキレバー入力   | 0.0〜1.0 |

**Bool チャンネル**

| CH | 型   | 用途                     |
|----|------|--------------------------|
| 1  | Bool | 右ドアボタン             |
| 2  | Bool | 左ドアボタン             |
| 3  | Bool | 室内ライトトグルボタン   |
| 4  | Bool | スポットライトトグルボタン |
| 5  | Bool | マスターリクエストボタン |
| 6  | Bool | パーキングブレーキトグルボタン |
| 7  | Bool | バック（逆転）トグルボタン |

## 出力チャンネル一覧

**Number チャンネル**

| CH | 型     | 用途         | 値域       |
|----|--------|--------------|------------|
| 1  | Number | スロットル出力 | -1.0〜1.0 |
| 2  | Number | ブレーキ出力   | 0.0〜1.0  |

**Bool チャンネル**

| CH | 型   | 用途                           |
|----|------|--------------------------------|
| 1  | Bool | 右ドア出力                     |
| 2  | Bool | 左ドア出力                     |
| 3  | Bool | 室内ライト出力                 |
| 4  | Bool | スポットライト出力             |
| 5  | Bool | テールライト出力               |

## 接続手順

一両車のため車間コネクタへの接続は不要です。運転台コントロールとアクチュエータをマイコンに接続してください。

```
[スロットルレバー]  ──> Number IN CH1 ┐
[ブレーキレバー]    ──> Number IN CH2 │
[右ドアボタン]      ──> Bool IN CH1   │
[左ドアボタン]      ──> Bool IN CH2   │
[室内ライトボタン]  ──> Bool IN CH3   ├── [マイコン (single_car.lua)]
[スポットボタン]    ──> Bool IN CH4   │
[マスターボタン]    ──> Bool IN CH5   │
[PBボタン]          ──> Bool IN CH6   │
[バックボタン]      ──> Bool IN CH7   ┘

[マイコン (single_car.lua)] ──> Number OUT CH1 ──> [スロットルアクチュエータ]
                            ──> Number OUT CH2 ──> [ブレーキアクチュエータ]
                            ──> Bool OUT CH1   ──> [右ドアアクチュエータ]
                            ──> Bool OUT CH2   ──> [左ドアアクチュエータ]
                            ──> Bool OUT CH3   ──> [室内ライト]
                            ──> Bool OUT CH4   ──> [スポットライト]
                            ──> Bool OUT CH5   ──> [テールライト]
```

## 操作手順

- **マスターリクエストボタン（Bool IN CH5）**: 押すたびにマスター ON/OFF が切り替わる。マスター OFF 時はスロットル=0・ブレーキ=100% に固定される。
- **スロットルレバー（Number IN CH1）**: マスター ON 時に有効。0〜100% の入力をスロットルに反映する。
- **ブレーキレバー（Number IN CH2）**: マスター ON 時に有効。0〜100% の入力をブレーキに反映する。
- **バックボタン（Bool IN CH7）**: 押すたびに逆転モード ON/OFF が切り替わる。ON 時はスロットルの向きが反転する。
- **パーキングブレーキボタン（Bool IN CH6）**: 押すたびに PB ON/OFF が切り替わる。ON 時はスロットル=0・ブレーキ=100% に固定される（レバー入力を無視）。
- **右ドアボタン（Bool IN CH1）**: 押すたびに右ドア開閉が切り替わる。
- **左ドアボタン（Bool IN CH2）**: 押すたびに左ドア開閉が切り替わる。
- **室内ライトボタン（Bool IN CH3）**: 押すたびに室内ライト ON/OFF が切り替わる。
- **スポットライトボタン（Bool IN CH4）**: 押すたびにスポットライト ON/OFF が切り替わる。

## 注意事項

- **マスターが OFF の間はブレーキ 100% が適用されます。** 発車前に必ずマスターボタンを押してください。
- **パーキングブレーキはブレーキレバーより優先されます。** PB が ON の間はレバーを操作しても速度は変わりません。
- **テールライトはマスター OFF 時に自動点灯します。** マスターを取得すると自動消灯します。手動操作はできません。
- **このスクリプトは一両車専用です。** 車間コネクタを他の車両に繋いでも信号は送受信されません。
- **ビークルをリスポーンするとすべての状態がリセットされます。** ドア・ライト・マスター状態はリスポーン後に再設定してください。

# 2. car_logic.lua + car_display.lua — 汎用列車コントローラー

## 概要

TA（前部運転台）・TB（中間車）・TAB（後部運転台）・一両車に対応する**汎用コントローラー**です。
車間通信でマスター（運転権）と運転命令を編成全体へ伝播し、複数両編成を 1 つの運転台から制御します。

マイコンの 8192 バイト上限を超えたため、**2 つのマイコンに分割**しています。

| マイコン | スクリプト | 役割 |
|---------|-----------|------|
| ロジック | `car_logic.lua` | 入力読取・マスター・車間通信・全制御ロジック・アクチュエータ出力 |
| 表示 | `car_display.lua` | ロジックの出力を受け取りモニタへ描画するだけ（onTick で受信・onDraw で描画） |

> 表示チップ（`car_display.lua`）は、ロジックチップの**ローカル出力コンポジットと同じバスに接続**します。専用配線は不要で、アクチュエータと同じ信号線にモニタ用マイコンをぶら下げるだけです。

## 対象車種

プロパティ `Is TA` / `Is TAB` の組み合わせで車種を決定します。**全車のロジックチップに同じ `car_logic.lua` を貼り付け**、プロパティのみ変えてください（`car_display.lua` にプロパティ設定は不要）。

| 車種 | Is TA | Is TAB | 説明 |
|------|-------|--------|------|
| TA   | true  | false  | 先頭車（前部運転台） |
| TB   | false | false  | 中間車（客車・運転台なし） |
| TAB  | false | true   | 最後尾車（後部運転台） |
| 一両車 | true | true   | 先頭・最後尾を兼ねる単体車両（車間通信なし） |

## プロパティ設定

| ラベル  | 型   | 説明              | デフォルト |
|--------|------|-------------------|-----------|
| Is TA  | Bool | 先頭車なら true   | false     |
| Is TAB | Bool | 最後尾車なら true | false     |

| 6  | Bool | パーキングブレーキトグルボタン |

## 接続手順

### 各マイコンのローカル接続

各車にロジックチップ（`car_logic.lua`）を 1 つ置きます。TA / TAB は運転台コントロールとアクチュエータを、TB はアクチュエータのみを接続します（TB に運転台入力は不要）。チャンネル割り当ては上記一覧の通りです。

### 表示チップの接続

運転台のあるTA / TABに、表示用マイコン（`car_display.lua`）を追加し、**ロジックチップのローカル出力（CH1〜）をモニタ用マイコンの入力へ分岐**します。さらにモニタ用マイコンの映像出力をモニタ部品へ繋ぎます。

```
[car_logic.lua] ──ローカル出力 CH1〜── ┬─→ [各アクチュエータ]
                                       └─→ [car_display.lua] ──→ [モニタ]
```

> 表示が不要な車（TB など）には `car_display.lua` を置かなくて構いません。

### 車間コネクタの連結

**前隣車の後方コネクタ（CH21〜）を、自車の前方コネクタ（CH11〜）へ対応付けて配線**します。編成全体では次のように繋がります。

```
   編成前方                                            編成後方
 ┌──────┐      ┌──────┐      ┌──────┐      ┌───────┐
 │  TA  │──┐ ┌─│  TB  │──┐ ┌─│  TB  │──┐ ┌─│  TAB  │
 └──────┘  │ │ └──────┘  │ │ └──────┘  │ │ └───────┘
   後方CH21〜┘ └CH11〜前方  後方┘ └前方   後方┘ └前方CH11〜
        (A後方 ↔ B前方 を対応付け)
```

- TA は後方コネクタのみ、TAB は前方コネクタのみを連結に使用します（編成端の外側コネクタは未接続でも問題ありません）。
- 各車のローカル出力（CH1〜5）、前方コネクタ（CH11〜）、後方コネクタ（CH21〜）はチャンネルが重複しないため、1 つのコンポジット入出力にまとめて配線できます。

## 操作手順

操作は TA / TAB の運転台で行います（TB に運転台はありません）。

- **マスターリクエストボタン（Bool IN CH5）**: 押すたびにマスター ON/OFF が切り替わる。ON にした運転台が編成全体の運転権を握る。
- **スロットルレバー（Number IN CH1）**: マスター取得中に有効。値が編成全車のスロットルへ伝播する。バック ON 時は向きが反転する。
- **ブレーキレバー（Number IN CH2）**: マスター取得中に有効。値が編成全車のブレーキへ伝播する。
- **バックボタン（Bool IN CH7）**: 押すたびに逆転モード ON/OFF が切り替わる。ON 時はスロットルの向きが反転する。
- **パーキングブレーキボタン（Bool IN CH6）**: 押すたびに PB ON/OFF が切り替わる。ON 時はブレーキ命令を 100% に固定して全車へ伝播する。
- **右ドアボタン（Bool IN CH1）**: 押すたびに右ドア開閉が切り替わり、全車へ伝播する。
- **左ドアボタン（Bool IN CH2）**: 押すたびに左ドア開閉が切り替わり、全車へ伝播する。
- **室内ライトボタン（Bool IN CH3）**: 押すたびに室内ライト ON/OFF が切り替わり、全車へ伝播する。
- **スポットライトボタン（Bool IN CH4）**: 押すたびにスポットライト ON/OFF が切り替わる。点灯するのは TA / TAB のみ。

## 注意事項

- **編成のマスターは常に 1 つです。** 別の運転台でマスターを取得すると、それまでのマスターは自動的に解除されます。
- **マスター不在の間は全車がスロットル=0・ブレーキ=100% になります。** 発車前にいずれかの運転台でマスターを取得してください。
- **連結配線は「前隣車の後方コネクタ ↔ 自車の前方コネクタ」で対応付けます。** 向きを誤ると命令やマスターが伝播しません。
- **全車両の「右ドア出力」を編成前方（TA 方向）基準で同じ物理側へ取り付けてください。** 本スクリプトはマスター方向フラグを使って左右を自動でマッピングするため、設置側が車両ごとにばらつくとドアの左右が逆になります。
- **後部運転台（TAB）でマスターを取ると、左右と前後が自動で反転します。** TAB のレバーを前方へ倒すと編成は TAB 側へ進み、ドアの左右も TA 基準へ正しく変換されます。
- **テールライト・スポットライトは TA / TAB のみ出力します。** テールライトは自車がマスターを保持していない間だけ自動点灯します。
- **方向マーカー（Number CH13/23）は車種判定に使われます。** TA は 1.0、TAB は 2.0 を発信し、TB はそれを反対側へ中継します。
- **一両車として使う場合は `Is TA` と `Is TAB` を両方 true にします。** この場合は車間通信を行わず、`single_car.lua` と同じ動作になります。
- **ビークルをリスポーンするとすべての状態がリセットされます。** ドア・ライト・マスター状態はリスポーン後に再設定してください。
