# UI 設計書 — 運転情報モニター

**バージョン**: 1.1.0
**作成日**: 2026-06-01
**更新日**: 2026-06-02
**対象**: 3×1 モニター（96×32 px）

---

## 1. 概要

TA・TAB の運転台に搭載する**3×1サイズモニター**に、運転に必要な情報をリアルタイム表示する。

- 解像度: **96 × 32 px**（1タイル = 32px）
- 描画関数: `onDraw()` 内で `screen.*` API を使用
- 情報源: `onTick()` で取得したグローバル変数を参照

## 3. 各領域の詳細設計

### 3.1 スロットル領域（x: 0〜31, y: 0〜31）

スロットルは `-1.0〜1.0` のため、ゲージ中央が 0（停止）、上半分が前進、下半分が後退を表す。
バックモード時はバー色を青に変更して視覚的に区別する。

| 要素 | 座標 | 内容 |
|------|------|------|
| ラベル | x=1, y=1 | `"THR"` テキスト（白） |
| バーゲージ（背景） | x=1, y=8, w=12, h=20 | 暗灰色 `(60,60,60)` |
| 中央ゼロ線 | x=1, y=18, w=12 | 白細線（0位置を示す） |
| バーゲージ（前進: thr>0） | x=1, y=18-10*thr, w=12, h=10*thr | 緑 `(80,200,80)` |
| バーゲージ（後退: thr<0） | x=1, y=18, w=12, h=10*(-thr) | 青 `(80,120,220)` |
| 数値 | x=15, y=14 | `math.floor(math.abs(thr)*100).."%"` テキスト（白） |
| バック表示 | x=15, y=22 | thr<0 のとき `"REV"` テキスト（青） |

### 3.2 ブレーキ領域（x: 32〜63, y: 0〜31）

| 要素 | 座標 | 内容 |
|------|------|------|
| ラベル | x=33, y=1 | `"BRK"` テキスト（白） |
| バーゲージ（背景） | x=33, y=8, w=12, h=20 | 暗灰色 `(60,60,60)` |
| バーゲージ（値） | x=33, y=8+h*(1-brk), w=12, h=20*brk | 赤 `(220,80,80)` |
| 数値 | x=47, y=14 | `math.floor(brk*100).."%"` テキスト（白） |

### 3.3 ステータス領域（x: 64〜95, y: 0〜31）

| 要素 | 座標 | 内容 |
|------|------|------|
| MASTER行 | x=65, y=1 | `"MST:"` + マスター名（TA/TAB/--） |
| 右ドアアイコン | x=65, y=10 | `"RD"` + 状態インジケータ |
| 左ドアアイコン | x=79, y=10 | `"LD"` + 状態インジケータ |
| 室内ライトアイコン | x=65, y=20 | `"LT"` + 状態インジケータ |
| スポットライトアイコン | x=79, y=20 | `"SP"` + 状態インジケータ |
| テールライトアイコン | x=65, y=27 | `"TL"` + 状態インジケータ（TA/TABのみ表示） |
| パーキングブレーキ | x=79, y=27 | `"PB"` + 状態インジケータ（赤） |

**状態インジケータ色:**
- ON: 塗りつぶし円 `(255,220,0)` （黄）
- OFF: 枠のみ円 `(100,100,100)` （灰）

## 4. 色定義

| 色名 | RGB | 用途 |
|------|-----|------|
| 背景 | `(15, 15, 15)` | 画面背景 |
| テキスト白 | `(220, 220, 220)` | 通常テキスト |
| テキスト黄 | `(255, 220, 0)` | アクティブ/強調 |
| テキスト赤 | `(255, 80, 80)` | 警告・ブレーキ |
| バー緑 | `(80, 200, 80)` | スロットルバー（前進） |
| バー青 | `(80, 120, 220)` | スロットルバー（後退） |
| バー赤 | `(220, 80, 80)` | ブレーキバー・PBインジケータ |
| ゲージ背景 | `(60, 60, 60)` | バーの背景 |
| 区切り線 | `(50, 50, 50)` | 領域間の縦線 |

---

## 5. 描画フロー（`onDraw()` 内）

```
1. screen.drawClear() — 背景クリア（デフォルト黒）
2. 背景色で全画面塗りつぶし
3. 区切り縦線を描画（x=32, x=64）
4. スロットルバーとラベル描画
5. ブレーキバーとラベル描画
6. マスター状態テキスト描画
7. ドア・ライト状態アイコン描画
```

## 6. 表示データとグローバル変数の対応

| 表示内容 | グローバル変数 | 型 |
|----------|--------------|-----|
| スロットル値 | `g_throttle` | Number (-1〜1) |
| ブレーキ値 | `g_brake` | Number (0〜1) |
| マスター名 | `g_master_label` | String ("TA"/"TAB"/"--") |
| 右ドア状態 | `g_door_right` | Bool |
| 左ドア状態 | `g_door_left` | Bool |
| 室内ライト状態 | `g_light` | Bool |
| スポットライト状態 | `g_spotlight` | Bool |
| テールライト状態 | `g_taillight` | Bool |
| パーキングブレーキ状態 | `g_parking_brake` | Bool |

---

## 7. 制約事項

- `onDraw()` 内でコンポジット入出力は使用不可
- グローバル変数は `onTick()` 内で更新し、`onDraw()` で参照する
- テキストフォントは固定幅（1文字 = 約4px）のため、長い文字列は切り詰める
