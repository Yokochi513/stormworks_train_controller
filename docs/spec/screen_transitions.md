# 画面遷移設計書 — 運転情報モニター

**バージョン**: 1.0.0
**作成日**: 2026-06-01

---

## 1. 概要

3×1モニターは常時単一の**運転情報画面**を表示する。画面切り替えは行わない。

ただし、マスター状態・ドア状態・ライト状態に応じて**表示内容が動的に変化する**。

---

## 2. 表示状態遷移

### 2.1 マスター表示の変化

```
[ NO_MASTER ]          [ TA_MASTER ]          [ TAB_MASTER ]
  MST: --                MST: TA                MST: TAB
  THR/BRK表示なし        THR/BRK 通常表示        THR/BRK 通常表示
  （全バーゼロ）
```

**遷移条件:**

| 遷移 | トリガー |
|------|---------|
| NO_MASTER → TA_MASTER | TA のマスターリクエストボタン ON |
| NO_MASTER → TAB_MASTER | TAB のマスターリクエストボタン ON |
| TA_MASTER → TAB_MASTER | TAB のマスターリクエストボタン ON |
| TAB_MASTER → TA_MASTER | TA のマスターリクエストボタン ON |
| TA_MASTER → NO_MASTER | （未定義 — マスター保持を継続） |

### 2.2 ドア状態インジケータ

```
右ドア閉:  RD ○（灰枠）
右ドア開:  RD ●（黄塗り）
左ドア閉:  LD ○（灰枠）
左ドア開:  LD ●（黄塗り）
```

**遷移条件:**

| 遷移 | トリガー |
|------|---------|
| 閉 → 開 | 右/左ドアボタン ON（トグル） |
| 開 → 閉 | 右/左ドアボタン ON（トグル） |

### 2.3 ライト状態インジケータ

```
室内ライト OFF:  LT ○（灰枠）
室内ライト ON:   LT ●（黄塗り）
スポットライト OFF:  SP ○（灰枠）
スポットライト ON:   SP ●（黄塗り）
```

**遷移条件:**

| 遷移 | トリガー |
|------|---------|
| OFF → ON | ライトトグルボタン ON |
| ON → OFF | ライトトグルボタン ON |

---

## 3. スロットル・ブレーキバー動的表示

バーの高さはリアルタイムに変化する（毎ティック更新）。

```
スロットル 0%:  ゲージ空（バーなし）
スロットル 50%: ゲージ半分（緑バー）
スロットル 100%: ゲージ全体（緑バー）

ブレーキ 0%:   ゲージ空
ブレーキ 100%: ゲージ全体（赤バー）
```

NO_MASTER 状態では、スロットル=0/ブレーキ=100% の表示（フルブレーキ）となる。

---

## 4. 状態管理まとめ

モニターの各表示要素に対応するグローバル変数と更新タイミング:

| 表示要素 | 変数 | 更新タイミング |
|----------|------|--------------|
| スロットルバー | `g_throttle` | 毎ティック（マスター入力 or 受信値） |
| ブレーキバー | `g_brake` | 毎ティック（マスター入力 or 受信値） |
| マスターラベル | `g_master_label` | マスター状態変化時 |
| 右ドアアイコン | `g_door_right` | ドアボタントグル時 / 受信時 |
| 左ドアアイコン | `g_door_left` | ドアボタントグル時 / 受信時 |
| 室内ライト | `g_light` | トグル時 / 受信時 |
| スポットライト | `g_spotlight` | トグル時 / 受信時 |
