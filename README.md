# stormworks_train_controller

汎用的な列車制御装置。

搭載する車両の種類は

- 先頭車 : TA
- 中間車(客車) : TB
- 最後尾車 : TAB
- 一両車 : TA, TAB

の4種類がある。

それぞれ
コントローラから出力される、出力チャンネルは以下

| 機能 | TA | TAB | TB |
|------|----|----|-----|
| スロットル(0〜1) | ✓ | ✓ | ✓ |
| ブレーキ(0〜1) | ✓ | ✓ | ✓ |
| 右ドア(Bool) | ✓ | ✓ | ✓ |
| 左ドア(Bool) | ✓ | ✓ | ✓ |
| 室内ライト(Bool) | ✓ | ✓ | ✓ |
| スポットライト(Bool) | ✓ | ✓ | — |

マスター（運転権）システム
- 最初は誰もマスターを持っていない
- TAかTABでリクエストを押すとマスターになる
- マスターは常に0か1（同時に2人はなれない）
- TAがリクエスト→TABのマスターが消える（逆も同様）
- マスターが切り替わってもBool入力の状態は維持

ドアの左右判定
- マスターを持つTA/TABの前方向を基準に左右を決める

通信・構成
- 各マイコンは隣の車両とのみ通信可（コンポジット信号）
- 前後コネクターの信号は内容次第で区別できる
- 車両の向きはコネクター位置から判断
- property.getBool('Is TA') でTA/TBを判別
- property.getBool('Is TAB') でTABを判別
- 3×1モニターにスロットル等の運転情報を表示
